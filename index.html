// Under the hood

How it works

01
🎥

Camera captures

Your browser accesses the webcam — completely local, nothing sent to a server.

02
🦴

MediaPipe detects

Google's MediaPipe Hands tracks 21 hand landmarks in real time at 30+ FPS.

03
🧠

AI classifies

A trained gesture classifier maps the landmark positions to ASL letters.

04
💬

Text appears

The recognized letter is added to your output with speak-aloud support.

// Project context

Built for inclusion

SignSpeak is an open-source AI project that makes communication more accessible. Built with MediaPipe Hands, a custom gesture classifier, and pure browser APIs — no backend required.

This project demonstrates real-time computer vision, machine learning inference in the browser, and accessible design — all in one.
